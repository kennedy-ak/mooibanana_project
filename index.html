Pressure Vessel Analysis Report

Numerical Solution using Thomas Algorithm

Date: October 19, 2025

1. Problem Statement

A thick pressure vessel with inner radius a and outer radius b is being tested in the laboratory to check its ability to withstand pressure. The differential equation for the radial displacement u of a point along the thickness is:

d²u/dr² + (1/r)(du/dr) - u/r² = 0
Boundary Conditions:
u(5) = 0.008731
u(8) = 0.0030769

2. Numerical Method

2.1 Discretization Parameters

  • Step length: h = Δr = 0.5
  • Domain: 5 ≤ r ≤ 8
  • Grid points: r = {5.0, 5.5, 6.0, 6.5, 7.0, 7.5, 8.0}
  • Number of unknowns: 5 interior points

2.2 Central Difference Approximations

du/dr = (ui+1 - ui-1) / (2Δr)

d²u/dr² = (ui+1 - 2ui + ui-1) / (Δr)²

2.3 Difference Equation

Substituting the approximations into the differential equation and simplifying yields:

ai ui-1 + bi ui + ci ui+1 = di

where:

ai = 1 - Δr/(2ri) = 1 - 0.25/ri
bi = -2 - (Δr)²/ri² = -2 - 0.25/ri²
ci = 1 + Δr/(2ri) = 1 + 0.25/ri
di = 0 (modified at boundaries)

3. Thomas Algorithm

The Thomas Algorithm (tridiagonal matrix algorithm) consists of two phases:

Forward Elimination:
c'0 = c0/b0, d'0 = d0/b0
For i = 1 to n-1:
  c'i = ci/(bi - aic'i-1)
  d'i = (di - aid'i-1)/(bi - aic'i-1)
Back Substitution:
un-1 = d'n-1
For i = n-2 to 0:
  ui = d'i - c'iui+1

5. Physical Interpretation

The solution demonstrates that the radial displacement decreases monotonically from the inner radius to the outer radius of the pressure vessel. This behavior is consistent with the physical expectations for a thick-walled pressure vessel under internal pressure:

  • Maximum displacement occurs at the inner radius (r = 5.0) where stress concentration is highest
  • Displacement gradually decreases toward the outer radius (r = 8.0)
  • The smooth variation indicates stable structural behavior under the applied pressure

6. Conclusion

The Thomas Algorithm successfully solved the discretized differential equation for radial displacement in the thick pressure vessel. The numerical solution satisfies both boundary conditions and exhibits physically reasonable behavior. The method proved efficient and accurate for this boundary value problem.
